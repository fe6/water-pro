自定义滚动条的容器

## 何时使用

- 统一不必要重复的样式
- 多地方使用，统一封装

## API

| 属性           | 说明                | 类型            | 默认值 |
| -------------- | ------------------- | --------------- | ------ |
| native         | 是否用原生滚动      | boolean         | false  |
| showHorizontal | 是否显示横向滚动条  | boolean         | false  |
| wrapStyle      | 滚动外框的样式      | string \| array | -      |
| viewStyle      | 滚动内容的样式      | string \| array | -      |
| wrapClass      | 滚动外框的 class 名 | string \| array | -      |
| viewClass      | 滚动内容的 class 名 | string \| array | -      |
