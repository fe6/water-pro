角标

## 何时使用

一些通用的三角，表示弹框状态。

## API

下面属性只能设置一个，设置多个不生效。

| 参数        | 说明     | 类型    | 可选值 | 默认值 | 版本 |
| ----------- | -------- | ------- | ------ | ------ | ------ |
| expand      | 向下方向 | boolean | | - | - |
| top  | 向上方向 | boolean | | - | - |
| bottom | 向下方向 | boolean | | - | - |
| inset| 向右     | boolean | | - | - |
| helpMessage | 文字提示 | string  | | - | - |
| size | 大小 | number  | | 16 | 4.0 |
| colors | 颜色 | string[] | | - | 4.0 |
